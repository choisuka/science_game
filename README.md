10가지 주제, 주제마다 5문제 퀴즈 도전.

**🌐 바로 가기:** https://choisuka.github.io/science_game

| 파일 | 설명 |

 └─ 주제 카드 클릭 → 유튜브 영상 소개 화면 → 퀴즈 (5문제) → 결과

## 기술 스택

- 순수 HTML + CSS + JavaScript (단일 파일, 빌드 없음)
- Web Audio API (별 배경 Canvas 애니메이션)
- YouTube iframe 임베드 (주제별 관련 영상)
- GitHub Pages 배포

---

## 유튜브 영상 교체 방법

`index.html` 의 TOPICS 배열에서 각 주제의 `vid` 값을 바꾸면 돼요.

```js
{ id:'space', vid:'여기에_유튜브_ID', ... }
```

유튜브 URL에서 `v=` 뒤 코드가 ID예요.
예: `youtube.com/watch?v=`**`ABC123xyz`** → `vid:'ABC123xyz'`

## 테마 변경

`preview.html` 을 브라우저로 열어서 원하는 테마 선택 후
`index.html` / `story.html` 의 `:root` CSS 변수를 교체하면 돼요.

```css
:root {
  --bg: #001219;   /* 배경 */
  --bg2: #0a3040;  /* 카드 배경 */
  --bg3: #0a9396;  /* 강조 배경 */
}
```
